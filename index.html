§1 - HTML

  • HTML - відповідає за наповнення веб-сторінки контентом, структуру і зміст.
  • CSS - використовується для оформлення та позиціонування, зовнішній вигляд.
  • JavaScript - дозволяє додати функціонал, реакцію на дії користувача.

Специфікація HTML Living Standard - головний документ, який описує стандарти, можливості і майбутній розвиток мови HTML.

Довідники:

  • Довідник HTML-тегів Для того щоб дізнатися доступні атрибути тегу та їх призначення

  • Довідник CSS-властивостей
  • Керівництво для написання коду в процесі вивчення технологій.
  1. <p> - універсальний контейнер для групування дрібних фразових елементів, відокремлення їх один від одного, і подальшої стилізації.
  2. <h1> - <h6> - визначає текстові заголовки смислових розділів різного рівня, які вказують на важливість секції контенту, розташованого після них.
  3. <li> - елементи списку, діти списку
  4. <ol> - створює нумерований (упорядкований) список, тобто кожен елемент списку пронумерований.
  5. <ul> - створює маркований (невпорядкований) список, кожен елемент якого починається з невеликого символу (маркера).
  6. <a> - призначений для створення посилань, тексту, клікаючи на який, переходимо на іншу сторінку, завантажуємо файл тощо.
    • href="адреса" - aдреса посилання задається в обов'язковому атрибуті
    • target="_blank" - визначає те, в який вкладці повинен відкриватися документ, на який веде посилання.
    • rel="noreferrer nofollow noopener" - відношення між поточним і пов'язаним документом
    • download="ім'я файлу" - повідомляє браузеру, що зазначений ресурс не повинен відкриватися, а завантажуватися в момент, коли користувач клікне на посилання.
    • href="tel:+14251234563" - здійснювати дзвінки на телефони
    • href="mailto:example@mail.ua - відправляти повідомлення
  7. <button> - Інтерактивний елемент, який оживляється за допомогою JavaScript.
    • type="button"
    • type="submit" - за замовчуванням
  8. <img> - призначений для розмітки зображень у різних графічних форматах.
    • src="шлях" - обов'язковий атрибут, вказує адресу зображення. Шлях до зображення може бути абсолютним, або відносним.
    • alt="опис" - обов'язковий атрибут, альтернативний опис.
    • width="значення" і height="значення" - задають розміри зображення в пікселях.
  9. <header> - Шапка (хедер) виділяє вступну частину всієї сторінки, складної секції або статті.
  10. <footer> - Підвал (футер) - це фінальна частина всієї сторінки, її розділу або статті.
  11. <main> - Основний (унікальний) вміст, що не повторюється на інших сторінках сайту.
  12. <nav> - Визначає розділ основної навігації з посиланнями на секції поточної або інших сторінок.
  13. <article> - Незалежна, відокремлювана, і така, що має самостійне значення, частина документа.
  14. <section> - Великий розділ, що об'єднує вміст за смисловим значенням.
  15. <div> - Універсальний контейнер без семантичного значення.
  16. <table> - визначає всю таблицю, контейнер для контенту.
  17. <tr> - рядок таблиці (table row). Не може бути порожнім, повинен містити хоча б одну клітинку.
  18. <td> - клітинка таблиці (table data). Можна використовувати тільки всередині рядка.
  19. <th> - (table header) - семантичний тег клітинки-заголовка.
  20. <thead> - шапка, обгортає групу рядків таблиці в області заголовка.
  21. <tbody> - тіло, обгортає основну групу рядків таблиці.
  22. <tfoot> - підвал, означає групу рядків таблиці в області нижнього колонтитула.
  23. <caption> - заголовок таблиці

  24. <dl> - це контейнер для списку пар термінів та їх описів, тобто словник.
  25. <dt> - для розмітки термінів
  26. <dd> - для описів
  27. <span> - Універсальний контейнер для дрібного текстового контенту
  28. <pre> - дозволяє

  30. <time> - для часу datetime="pppp-мм-дд" - атрибут - 14 грудня 2019 року.
  31. <address> - для зберігання контактної інформації про автора
  32. <strong> і <b> - роблять текст жирним <strong> - семантичний
  33. <em> і <i> - роблять текст курсивним <em> - семантичний
  34. <video> - Для розмітки відеоконтенту
    • src - вказує адресу відеофайлу

    • width і height - задають висоту і ширину як у зображення
    • controls - атрибут-прапорець, якщо присутній, додаються елементи управління відео

    • loop - атрибут-прапорець, якщо присутній, аудіо буде повторюватися в циклі
    • preload вказує режим попереднього завантаження аудіофайлу
    • none - не завантажувати аудіо заздалегідь
    • metadata - завантажити службову інформацію (тривалість тощо)
    • auto - заздалегідь завантажувати увесь аудіофайл

§2 Селектори та колір, Текст та шрифт

Оформлення тексту

Властивість text-decoration - Задає різні декоративні підкреслення та інші ефекти тексту.

Властивість text-transform - Управляє регістром символів у тексті

Властивість text-align - Управляє вирівнюванням текстового вмісту елемента за горизонталлю.

Властивість text-indent - Встановлює величину відступу першого рядка блоку тексту

Властивість line-height - Встановлює міжрядковий інтервал

Властивість letter-spacing - Визначає інтервал між символами

Властивість word-spacing - Визначає інтервал між словами

Властивість text-shadow - Контролює тінь тексту, а також встановлює її параметри: колір, зміщення і радіус розмиття.

Властивість white-space - як і тег <pre>, але гнучкіше управляє форматуванням пробілів і перенесень тексту.

Властивості шрифту

Властивість font-size - Задає розмір шрифту елемента в абсолютних або відносних одиницях виміру

Властивість font-weight - Контролює товщину накреслення (жирність, вага) літер шрифту.

Властивість font-style - Задає накреслення тексту, наприклад можна зробити текст похилим.

Властивість font-family - задає шрифт тексту елемента.

Псевдоелемент ::first-letter - 

Управляє стилем першого символу в тексті елемента

Псевдоелемент ::selection - Управляє стилем виділення тексту елемента, перевизначаючи стандартні значення з таблиці стилів браузера.

§3 Блокова модель, Flexbox

  • Властивості width і height - Ці властивості дозволяють явно вказати загальну ширину і висоту елемента
  • Властивість box-sizing - Встановлює тип блокової моделі елемента - формулу розрахунку його розмірів.

  • Властивість padding - внутрішні поля між контентом і рамкою для усіх сторін елемента.

  • Властивість margin - зовнішні відступи від рамки для всіх сторін елемента.

  • Властивість border - контролює ширину, стиль і колір рамки (межі) елемента.

  • Bластивість border-radius - Для того щоб закруглити всі кути рамки

  • Властивість display: block - Візуально - це прямокутні області, що йдуть одна за одною зверху вниз.

  • Bластивість display: inline - Візуально розташовуються в одному рядку один за одним доти, доки в ньому вистачає місця, після чого почнуть перестрибувати на наступний рядок і заповнювати його.

  • Bластивість display: inline-block - Гібрид між блоковими і рядковим елементами, який взяв все найкраще від обох.

  • Bластивість display: none - дозволяє повністю приховати елемент, звільнивши його простір для інших.

  • Псевдокласи :first-child і :last-child - Дозволяють вибрати перший або останній елемент в колекції сусідів.

  • Псевдоклас :not(selector) - Дозволяє вибрати всі елементи, які не задовольняють критерій.

  • Псевдоклас :nth-child(an+b) - Вибирає елементи в колекції сусідів за номером, зазначеним в дужках, за допомогою циклу an+b, який дозволяє задати правило для послідовності елементів.

  • Псевдоклас :nth-last-child(an+b) - Аналог :nth-child() з відмінністю у тому, що відлік ведеться з кінця колекції (Останнього елемента).

  • Bластивість display: flex | inline-flex - Створює flex-контейнер, блоковий або рядково-блоковий, залежно від заданого значення, і встановлює flex-контекст для всіх дітей (не нащадків) контейнера, перетворюючи їх у flex-елементи.

  • Властивість flex-shrink - Визначає здатність елемента, у разі потреби, займати менше місця (стискатися), ніж його початковий розмір.

§4 Декоративні ефекти та елементи, Векторна графіка

  • Властивість background-color - Задає колір фону елемента.

  • Властивість background-image - Задає фонове зображення елемента, адреса якого вказується аргументом функції url().

  • Властивість background-repeat - Управляє повторенням фонового зображення.

  • Властивість background-position - Управляє положенням фонового зображення щодо меж елемента, використовуючи дві координати - x за горизонталлю та y за вертикаллю.

  • Властивість background-size - Ця властивість дозволяє підкоригувати розміри зображення, вказавши необхідну ширину і висоту.

    • Значення contain і cover - Вони намагаються максимально вмістити в контейнер фонове зображення, тільки одне - за рахунок обрізання зайвого, а інше - за рахунок появи порожнього простору в контейнері.

  • Властивість background-attachment - можна зафіксувати фон на місці і заборонити прокручуватися зі скролом.

  • Властивість object-fit - Визначає, яким чином вміст img буде вміщуватися в контейнер, якщо їх розміри або пропорції не збігаються.

  • Властивість object-position - Доповнює властивість object-fit, дозволяючи задати положення зображення в контейнері по осях X і Y.

  • Псевдоелементи ::before і ::after - використовуються для додавання декоративних ефектів (іконок, плашок, оверлеїв) без необхідності створення додаткових порожніх тегів в розмітці.

  • background-image: repeating-radial-gradient - ( circle, purple, purple 10px, #4b026f 10px, #4b026f 20px );

  • Властивість box-shadow - задає елементу одну або більше тіней. За замовчуванням розмір тіні збігається з розміром елемента, а її колір відповідає кольору його тексту.

  • Елемент <svg> - Визначає полотно, на якому можна малювати фігури.

  • Елемент <rect> - Створює прямокутну область.

  • Елемент <circle> - Створює коло, радіус якого задається атрибутом r.

  • Елемент <ellipse> - Створює еліпс.

  • Елемент <line> - Створює пряму лінію.

  • Елемент <polygon> - Описує багатокутник, вид якого визначається атрибутом points - набором точок у вигляді координат ліній, які розділені на групи.

  • Елемент <path> - Універсальний елемент для представлення фігур будь-якої складності.

  • Властивість fill - Управляє кольором заливання.

  • Властивість fill-opacity - Управляє тільки прозорістю кольору заливання, а не всього кольору або елемента, тобто відрізняється від opacity або stroke-opacity.

  • Властивість stroke - Встановлює колір обведення.

  • Властивість stroke-width - Контролює товщину обведення.

  • Властивість stroke-opacity - Управляє тільки прозорістю кольору обведення, а не всього кольору або елемента.

§5 Позиційовані елементи, Анімація

  • Властивість position - Змінює тип позиціонування елемента.

  • Властивість z-index - контролює глибину (порядок) позиціонованих елементів на сторінці по осі z - від очей користувача і до екрану.

  • Властивість overflow - управляє тим, як поводиться вміст елемента, якщо його розмір перевищує допустиму довжину або ширину.

  • Властивість transition-property - Задає властивості, до яких буде застосований ефект переходу.

  • Властивість transition-duration - задає проміжок часу, протягом якого повинен відбутися перехід.

  • Властивість transition-timing-function - Задає функцію розподілу часу, яка описує швидкість переходу властивості від одного значення до іншого за час, вказаний в transition-duration, тобто поведінка анімації.

  • Властивість transition-delay - Затримка, після якої почнеться анімація переходу.

  • Оголошення анімації @keyframes ім'яАнімації - дозволяє описати набір кадрів (frames, станів) анімації, яких повинно бути як мінімум два (початковий і кінцевий).

  • Властивість animation-name - Щоб використовувати вже оголошену анімацію.

  • Властивість animation-duration - Встановлює тривалість анімації - час, за який будуть пройдені всі кадри.

  • Властивість animation-timing-function - Встановлює функцію розподілу часу для анімації.

  • Властивість animation-delay - Встановлює затримку перед початком анімації в секундах або мілісекундах.

  • Властивість animation-iteration-count - Встановлює кількість повних повторень анімації.

  • Властивість animation-direction - Контролює напрямок відтворення анімації.

  • Властивість animation-fill-mode - Визначає, що відбувається зі стилями елемента до початку анімації і після її завершення.

  • Властивість animation-play-state - Дозволяє призупинити відтворення анімації.

  • Функція perspective() - Це ще одна функція трансформації для властивості transform, яка дозволяє задати перспективу одному елементу, до якого застосовується.

  • Властивість perspective - Дозволяє створити однакову перспективу цілій групі елементів на сцені.
